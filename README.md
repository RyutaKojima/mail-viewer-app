# Mail Viewer

## App

[Mail Viewer](https://mail-viewer-app.vercel.app/)

## How to develop

```shell
git clone git@github.com:RyutaKojima/mail-viewer-app.git mail-viewer-app

nodenv install

yarn install

yarn dev
```

formatting

```shell
yarn format
```
